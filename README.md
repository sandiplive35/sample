# Sandeep Dolai — Thumbnail Designer Portfolio

A premium, responsive portfolio website for a YouTube thumbnail designer.

## Positioning

The site presents Sandeep as a strategic thumbnail designer focused on clarity, curiosity, mobile readability and visual storytelling. All showcased pieces are clearly labelled as concept studies, so the portfolio does not invent client results, testimonials or performance claims.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Deploy

This is a static site and can be deployed directly to GitHub Pages, Vercel or Netlify.
